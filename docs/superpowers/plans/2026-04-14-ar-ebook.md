## Task 1: Content Extraction Pipeline

**Files:**
- Create: `scripts/extract-doc.ps1`
- Output: `content/ch{1,2,3}.txt` (UTF-8), `content/images/ch{1,2,3}/image*.png`

- [ ] **Step 1:** Write `scripts/extract-doc.ps1`. Uses Word COM. For each of 3 docs: `SaveAs2` with format `7` (wdFormatUnicodeText) → `content/ch{n}.txt`, and export document images by saving as `.docx` temp then unzipping `word/media/` to `content/images/ch{n}/`.

```powershell
$ErrorActionPreference = 'Stop'
$root = Split-Path -Parent $PSScriptRoot
$src = Join-Path $root 'บทที่ {0}_AR.doc'
$outTxt = Join-Path $root 'content\ch{0}.txt'
$outImg = Join-Path $root 'content\images\ch{0}'
$tmpDocx = Join-Path $env:TEMP 'ar_ch{0}.docx'

New-Item -ItemType Directory -Force -Path (Join-Path $root 'content\images') | Out-Null

$word = New-Object -ComObject Word.Application
$word.Visible = $false
try {
  1..3 | ForEach-Object {
    $n = $_
    $in = $src -f $n
    $txt = $outTxt -f $n
    $imgDir = $outImg -f $n
    $docx = $tmpDocx -f $n
    New-Item -ItemType Directory -Force -Path $imgDir | Out-Null

    $doc = $word.Documents.Open($in, $false, $true)
    $doc.SaveAs2($txt, 7)            # wdFormatUnicodeText (UTF-16 LE)
    $doc.SaveAs2($docx, 16)          # wdFormatDocumentDefault (.docx)
    $doc.Close()

    # unzip .docx, copy word/media/* → imgDir
    $extractDir = "$docx.extract"
    if (Test-Path $extractDir) { Remove-Item -Recurse -Force $extractDir }
    Expand-Archive -Path $docx -DestinationPath $extractDir -Force
    $media = Join-Path $extractDir 'word\media'
    if (Test-Path $media) {
      Copy-Item -Path (Join-Path $media '*') -Destination $imgDir -Force
    }
    Remove-Item -Recurse -Force $extractDir, $docx
    Write-Host "ch$n done"
  }
}
finally {
  $word.Quit() | Out-Null
  [System.Runtime.Interopservices.Marshal]::ReleaseComObject($word) | Out-Null
}
```

  pages: write
  id-token: write

  group: pages

    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4

    needs: build
    runs-on: ubuntu-latest
    environment:

      url: ${{ steps.deployment.outputs.page_url }}
    steps:
      - id: deployment
        uses: actions/deploy-pages@v4
